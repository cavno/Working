恒等式坍缩自由度　→　两杆操作(出价 · 结构)　→　机制模型(拍卖 · 坑位 · 变体 · 类型)　→　盈亏分账(投资型 / 盈利型 / 纯利型)。

R1 归因口径　广告归因销售滞后回填(SP 默认 7 天,SB/SD 可到 14 天),花费当日 → 近端 ACOS/CPA/TACOS 一律虚高;数据须陈化 7–14 天再判。
R2 平台会变　建议竞价算法、竞价乘子上限、广告位归类、COSMO、组合预算行为随版本调整;本图为 2024-08 快照,数字以当下实测为准。
R3 单调性假设　a/b/c/d 依赖"竞价↑→位置↑→PP 占比按结构变";低权重老链接 / 乱投放会破坏,此时不适用。
R4 拍卖为简化　adrank=bid·相关性、扣费=下家 adrank/你相关性+0.01 是 GSP 式简化,含 pacing / 预算感知出价等,不可精确验。
R5 常数需标定　PP×5–15、CVR 15/10/7/2–5%、降价 ≤5%/次 等为经验起点,不同类目 / 产品自校。
R6 合规红线　讲价 / 结构优化 / 否定 / 漏斗属正常运营;虚假订单 / 测评 / 刷坑产违反 Amazon ToS、可能触及法规,本图不含此类手法。
