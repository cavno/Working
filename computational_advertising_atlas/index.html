一个目标函数(带约束的整体利润最大化)+ 一个分解恒等式(eCPM = μ·ν)+ 对每个因子的三问(谁估计 · 说真话 · 估准)= 全书。第 3 版的统一钥匙是计算分工:CPC / 程序化 / 智能投放 三形态,是 eCPM 计算在供需两侧的三种切分。

恒真层　eCPM=μ·ν 是定义分解;"计费=分工"、"三形态=三切分"是市场结构分类学;不依赖行为假设,可直接用。
机制模型层　GSP/VCG/迈尔森的最优性依赖理性广告主、对称纳什均衡、μ⊥位置/ν⊥广告等简化;二价"说真话"在预算约束/自动出价下并不严格成立(书自认 oCPX 一价二价的模糊)。
工程近似层　可分解性假设(A2,书自认不合理)、CPC 的 ν(a) 统一报价(式5.6)、价格挤压 κ、MRP、平滑系数,皆经验量、需本地标定。
时代边界　第 3 版(2022)以 cookie / 第三方 DMP 为估计地基;隐私合规(GDPR / 后 cookie / 联邦学习)正抽掉地基 → μ、ν 可估性下降 → 计算分工天平进一步右移(供给方 / 第一方数据)。生成式创意、大模型语义检索/出价是书后方向;知识截止 2026-01,当下进展需联网核实。
