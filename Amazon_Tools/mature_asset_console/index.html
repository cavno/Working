A1 生命周期档位判定

A3 目标 ACOS 区间 [m·b, m]

ln-ln 弹性模型下利润最大化一阶条件：S=A·x^b，π=mS−x，dπ/dx=0 ⇒ 最优 ACOS = m·b。区间下限 m·b 是纯短期利润最大化（收割档贴这里），上限 m 是平衡线（防御 / 再加速档可贴近甚至短期越过）。带上的刻度：档位锚点、非品牌 ACOS、综合 ACOS——老品健康度看非品牌那根刻度。
