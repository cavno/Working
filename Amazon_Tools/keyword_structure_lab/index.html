结论一 · 纵向漏斗（绝对量）

M1 杜邦归因 —— 销量变化落在哪个因子

Δln(销售额) = Δln(曝光) + Δln(CTR) + Δln(CVR) + Δln(客单价)，归因穷尽且互斥。诊断映射：曝光 → 出价/预算/相关性；CTR → 搜索结果页素材的相对位次；CVR → 详情页与搜索意图错配或比价失败；AOV → 成交结构（变体/多件装）。

等待「本期 + 上期」两份搜索词报告
结论一 · 相对化（SQP 市场份额）

M2 SQP 双比率四象限 —— 与市场均值比，漏斗在哪层强、哪层弱

R1 = 点击份额 ÷ 曝光份额（>1 = SERP 素材强于市场）；R2 = 购买份额 ÷ 点击份额（>1 = 详情页转化强于市场）。价差列 = (品牌点击中位价 − 市场点击中位价) ÷ 市场价，超过 +10% 标记比价失败嫌疑。

结论二 · 横向组合结构

M3 词组合 = 持仓组合 —— 集中度 · 意图分布 · 花费-销售错配

HHI > 2500 视为高集中（单词依赖风险）；错配 = 花费份额 − 销售份额，正差是补贴/失血词，负差是提款机。中央散点是操作地图：x 点击量（对数）、y 收缩后 CVR、气泡 = 花费、颜色 = ACOS（绿→金→赤陶随恶化）。

结论三 · 统计纪律

M4 收缩估计 —— 小样本 CVR 是噪音，先收缩再决策

按意图类别做经验贝叶斯收缩：收缩CVR = (订单 + μν) ÷ (点击 + ν)，先验均值 μ 取类别汇总转化率，先验强度 ν 用矩估计自动拟合（可覆写）。任何加价/杀词决策只认「收缩CVR」列；护栏表列出裸眼会犯的错。

结论四 · 时间结构

M5 TACOS-ACOS 飞轮 · 季节 regime · 增量性

每月一行：月份, 总销售额, 广告销售额, 广告花费（逗号分隔）。TACOS 趋势性下降而 ACOS 持平 = 广告在沉淀自然排名资产（飞轮）；两者同步高位 = 纯依赖。礼品词占比表盘用于监控 Q4 regime 切换。

至少 4 期数据后给出飞轮判读
品牌词增量性实验协议（线下动作，非工具计算）
挑连续两周流量平稳期（避开促销/节日）：第一周正常投放作基线；第二周将品牌词广告组出价下调 50%（或暂停 Exact 品牌词组），其余不动。对比两周的总销量（广告 + 自然，取业务报告口径）差分：若总量不掉而广告销量下降，差额即被自然位回收的部分——那部分品牌词广告费买的是左手倒右手。玩具类目建议 Q1–Q2 淡季做，Q4 不做实验。
结论五 · 词是微市场

M6 修饰词显微镜 —— 市场给你的定位投票

从搜索词中剥离品类词根后，按年龄 / 人群 / 场景 / 主题 / 规格 / 功能聚合修饰词的流量与转化。销售额头部修饰词 = 市场对产品的心智定位；CVR 显著高于整体的修饰词应写进标题首屏；高流量低 CVR 的修饰词 = 定位错配，要么改 listing 要么否定。
